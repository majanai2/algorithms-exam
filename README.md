# Algorytmy – projekt egzaminacyjny

Repozytorium zawiera implementacje algorytmów
przygotowanych na egzamin z informatyki.

## Zawartość:
- Sito Eratostenesa – liczby pierwsze
- Płatek Kocha – fraktal rekurencyjny
- Schemat Hornera – obliczanie wielomianów
- Algorytm Euklidesa – NWD wielu liczb
- Faktoryzacja – rozkład na czynniki pierwsze

Każde zagadnienie zawiera:
- opis algorytmu
- implementację w języku Python
