# Zagadnienie 2 – Płatek Kocha

## Opis
Program generuje fraktal geometryczny – płatek Kocha –
z wykorzystaniem rekurencji.

Użytkownik może ustawić poziom głębokości rekurencji,
co wpływa na szczegółowość fraktala.

## Cechy fraktala
- samopodobieństwo
- rekurencyjna budowa
- nieskończona złożoność

## Zastosowania
- grafika komputerowa
- wizualizacje
- symulacje przyrodnicze
